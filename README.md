# MTProto Proxy с Fake TLS 🚀

Этот проект позволяет быстро поднять свой собственный прокси-сервер для Telegram (MTProto Proxy + Fake TLS) буквально за 1 минуту. Управление прокси происходит через удобный графический bash-интерфейс `tg-ui`.

## Установка и запуск

Скрипт можно запустить одной командой в вашем терминале (Linux/Windows WSL). Скопируйте и выполните этот код:

```bash
bash <(curl -sL https://raw.githubusercontent.com/lyfreedomitsme/MTProtoFakeTLS/main/install.sh)
```

Либо, если вы скачали этот репозиторий локально, перейдите в папку проекта и введите:
```bash
sudo ./install.sh
```

## Как пользоваться

После первой установки, в вашей системе появится команда:
```
tg-ui
```
Просто напишите `tg-ui` в консоли на сервере, чтобы открыть панель управления!

### В `tg-ui` доступны следующие возможности:
1. **Запустить / Перезапустить прокси**
2. **Изменить Fake TLS домен:** вы можете в любой момент изменить сайт маскировки и скрипт сам автоматически пересоздаст ключи и перезапустит контейнер.
3. **Изменить порт**
4. **Показать ссылку на подключение:** скрипт выдаст готовую ссылку вида `tg://proxy?...`
5. **Остановить и удалить контейнер с прокси**
6. **Посмотреть логи:** если что-то пошло не так, вы всегда сможете увидеть последние 20 логов Docker контейнера.
7. **Выход**

## Поддержка операционных систем

- **Ubuntu, Debian, CentOS, AlmaLinux** - 100% совместимо, скрипт автоматически проверит наличие Docker и установит его.
- **Windows / Windows Server** - работает прекрасно через встроенную подсистему WSL2 или если вы используете Git Bash вместе с запущенным Docker Desktop. Одной команды `bash <(...)` достаточно.
